Display the current Bitfinex Iota Price in Realtime. The Price updates automatically.

== Description ==

Just insert the shortcode `[iota_price]` wherever you want to display the current Iota price in realtime.

Data Source: [Bitfinex Websocket](wss://api.bitfinex.com/ws/)

**Features of this Plugin**

* Realtime Price
* Compatible with every Theme
* Easy to use
* Customizable Colors


= The Shortcode =

In order to embed the Iota Price on a new post/page and use the following shortcode:

`[iota_price]`

= Customizations (Optional) =

Your can force custom size and color of the price.

Example: `[iota_price size="24px" color="#d3d3d3"]`


== Installation ==

1. Upload `bitfinex-iota-price` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin from Admin > Plugins menu.
3. Once activated your site is ready!
